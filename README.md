### Building

Make sure you have installed the dependencies:

  * `python` 3
  * `git`
  * `rust`

Clone the [source] with `git`:
 ```sh
   git clone https://github.com/dedsecubr/pyrust-lang
   cd pyrust-lang
   ```

[source]: https://github.com/dedsecubr/pyrust-lang

And execute.
